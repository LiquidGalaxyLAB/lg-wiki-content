## Overview
_**What is an API Key?**_  
  
→ An application programming interface (API) key is a unique code or password that allows one software application to talk to another. It's a way for developers to control access to their web services or data by providing a unique key to users or other programs.  
  
→ It is used for;  
  
1 - Access Control  
2 - Act as a form of identification  
3 - enabling secure communication, and for many more usages  
  

* * *

  
_**Use of Google Maps API key:**_  
  
→ With a Google Maps API key, you can embed interactive maps on your Liquid Galaxy Apps, allowing your apps to access the Google Maps integration features.
