**What is UML?**  
  
It is a general-purpose visual modeling language that is intended to provide a standard way to visualize the design of a system. (It’s a set of rules specifically for drawing diagrams)  
  
**Refer to this link for example:** [Examples of UML diagrams - use case, class, component, package, activity, sequence diagrams, etc. (uml-diagrams.org)](https://www.uml-diagrams.org/index-examples.html)  
  
**I found these cool online tool, maybe it’s helpful for you:** [Online UML Diagram Tool (visual-paradigm.com)](https://online.visual-paradigm.com/diagrams/features/uml-tool/)
